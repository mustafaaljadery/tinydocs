# Tensor.transpose()

Returns the transposed tensor. The axes are swapped according to the values given in the `axes` argument.

## Usage
```python copy
from tinygrad.tensor import Tensor

a1 = Tensor([[1, 2, 3], [3, 4, 5]])
a1 = a1.transpose(0, 1)

print(a1.numpy())
```

## Return value
```bash
[[1. 3.]
 [2. 4.]
 [3. 5.]]
```
