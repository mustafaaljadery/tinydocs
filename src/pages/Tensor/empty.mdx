# Tensor.empty()

Creates a tensor with the specified size, without initializing entries.

[Look at code](https://github.com/tinygrad/tinygrad/blob/master/tinygrad/tensor.py#L122)

## Usage

```python copy
from tinygrad.tensor import Tensor

# init an empty tensor with shape (2, 3)
x = Tensor.empty(2, 3)

# print out the tensor values as a numpy array
print(x.numpy())
```

## Return Value

```bash
[[0. 0. 0.]
 [0. 0. 0.]]
```
