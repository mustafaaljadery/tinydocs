# Tensor.cos()

Returns the cosine of the specified tensor, element-wise.

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor([1, 2, 3, 4, 5, 6])
tensor = tensor.cos()

print(tensor.numpy())
```

## Return value
```bash
[ 0.54030234 -0.41614679 -0.9899925  -0.6536438   0.28366202  0.96017027]
```
