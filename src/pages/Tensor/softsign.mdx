# Tensor.softsign()

Returns the softsign of the elements of the input tensor element-wise.

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor([-2, -3, -2, -1, 0, 6])
tensor = tensor.softsign()

print(tensor.numpy())
```

## Return value
```bash
[-0.6666667 -0.75      -0.6666667 -0.5        0.         0.8571429]
```
