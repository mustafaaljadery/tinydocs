# Tensor.cat()

Returns the concatenation of the given sequence of seq tensors in the given dimension. All tensors must either have the same shape (except in the concatenating dimension) or be empty.

## Usage
```python copy
from tinygrad.tensor import Tensor

a1 = Tensor([[1, 2, 3]])
a2 = Tensor([[1, 2, 3]])

a1 = a1.cat(a2)


print(a1.numpy())
```


## Return value
```bash
[[1. 2. 3.]
 [1. 2. 3.]]
```
