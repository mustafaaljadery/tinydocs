# Tensor.manual_seed()

Sets the seed for generating random numbers for all devices.

## Usage

```python copy
from tinygrad.tensor import Tensor

# set the seed
Tensor.manual_seed(1337)
x = Tensor.rand(2, 3)

# set the seed again
Tensor.manual_seed(1337)
y = Tensor.rand(2, 3)

# print the same random tensors
print(x.numpy())
print(y.numpy())
```
