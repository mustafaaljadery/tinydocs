# Tensor.celu()

Returns the element-wise Celu activation function applied to the input tensor.

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor([-1, -5, 1, 2, 3, 4, 5])
tensor = tensor.celu()

print(tensor.numpy())
```

## Return value
```bash
[-0.63212055 -0.99326205  1.          2.          3.          4.
  5.        ]
```
