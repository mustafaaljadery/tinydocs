# Tensor.softplus()

Returns the softplus of the input tensor element-wise.

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor([-2, -3, -2, -1, 0, 6])
tensor = tensor.softplus()

print(tensor.numpy())
```

## Return value
```bash
[0.12692805 0.04858733 0.12692805 0.31326166 0.6931472  6.0024753 ]
```
