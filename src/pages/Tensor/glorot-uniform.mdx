# Tensor.glort_uniform()

A method for weight initialization of a tensor with a uniform distribution. 

[Paper link](https://proceedings.mlr.press/v9/glorot10a/glorot10a.pdf)

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor.glorot_uniform(3, 3)

print(tensor.numpy())
```
