# Tensor.slice()

Returns the slice of the tensor.

## Usage
```python copy
from tinygrad.tensor import Tensor

a1 = Tensor([[1, 2, 3], [4, 5, 6], [7, 8, 9]])
a1 = a1.slice([(0, 2), (1, 0)], 0)

print(a1)
```
