# Tensor.log2()

Returns the base-2 logarithm of the input tensor, element-wise.

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor([-1, -5, 1, 2, 3, 4, 5])
tensor = tensor.log2()

print(tensor.numpy())
```

## Return value
```bash
[      nan       nan 0.        1.        1.5849625 2.        2.321928 ]
```
