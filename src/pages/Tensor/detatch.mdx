# Tensor.detatch

Returns the same Tensor detached from the computation graph and with the `required_grad` flag set to `False`.

## Usage
```python copy
from tinygrad.tensor import Tensor

# create a tensor
tensor = Tensor([[1, 2, 3, 4, 5]])

# detatch tensor
print(tensor.detach())
```

## Return value
