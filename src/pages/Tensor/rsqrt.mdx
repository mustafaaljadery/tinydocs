# Tensor.rsqrt()

Returns the reciprocal of the square-root of each element of the input tensor, element-wise.

## Usage
```python copy
from tinygrad.tensor import Tensor

a1 = Tensor([[1, 2, 3], [3, 4, 5]])
a1 = a1.rsqrt()

print(a1.numpy())
```

## Return value
```bash
[[1.         0.70710677 0.57735026]
 [0.57735026 0.5        0.4472136 ]]
```
