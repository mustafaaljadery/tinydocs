# Tensor.permute()

Changes the order of dimensions in a tensor.

## Usage

```python copy
from tinygrad.tensor import Tensor

tensor = Tensor.randn(1, 2, 3)

print(tensor.numpy())

tensor = tensor.permute(2, 1, 0)

print(tensor.numpy())
```
