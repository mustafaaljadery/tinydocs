# Tensor.to()

Moves the Tensor from CPU to GPU or vice versa.

## Usage
```python copy
from tinygrad.tensor import Tensor

# create a tensor
tensor = Tensor([[1, 2, 3, 4, 5]])

# move my tensor to CPU
tensor = tensor.to("CPU")

# print the tensor
print(tensor)
```
