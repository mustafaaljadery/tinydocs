# Tensor.sign()

Returns an element-wise indication of the sign of a number. -1 for negative, 0 for zero, 1 for positive.

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor([-2, -3, -2, -1, 0, 6])
tensor = tensor.sign()

print(tensor.numpy())
```

## Return value
```bash
[-1. -1. -1. -1.  0.  1.]
```
