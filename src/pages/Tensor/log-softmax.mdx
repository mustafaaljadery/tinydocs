# Tensor.log_softmax()

Returns the log of softmax of the elements of the input tensor.

## Usage
```python copy
from tinygrad.tensor import Tensor

tensor = Tensor([-1, -5, 1, 2, 3, 4, 5])
tensor = tensor.log_softmax()

print(tensor.numpy())
```

## Return value
```bash
[ -6.4535193 -10.45352    -4.4535193  -3.4535196  -2.4535196  -1.4535195
  -0.4535195]
```
